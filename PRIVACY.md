# Privacy Policy

**spec-driven-development** is a local Claude Code / Cowork plugin consisting entirely of skill instructions (markdown files). It has no MCP servers, makes no network requests, and collects no user data of any kind.

All work performed by this plugin happens locally within your Claude session. No information is transmitted to any third party by this plugin.

If you have questions, open an issue at https://github.com/GameFixxer/spec-driven-development.
